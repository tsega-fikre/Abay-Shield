# Security Policy for Abay Shield

## Responsible Use
Abay Shield is a password security analysis tool intended strictly for:
- Ethical hacking
- Penetration testing on systems you own
- Educational purposes

**Illegal or unauthorized use is prohibited.**

## Reporting Vulnerabilities
If you find a security flaw in Abay Shield:
- Email: your-email@example.com  
- Do not publicly disclose until a fix is released.
